# Draughts10x10
Java Swing International Draughts with AI

Play against Computer (board 10x10)

White or Black

AI 1-5 -> 1 level = 2 moves (alfa & beta). AI uses bitboards (bit 0-49).

Hint Moveable Pieces (on/off)

Undo Move (turn=player or gameover)

Rotate Board

Animation (move, promotion, capture)
