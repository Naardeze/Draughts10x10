# Draughts10x10
Java Swing International Draughts with AI

Play draughts against computer (minimax with alfa beta pruning, using bitboards for extra performance).

No draw implemented!!

-White or Black

-AI 1-5 -> 1 level = 2 moves (alfa & beta)

-Undo move

-Hint On/Off (moveable pieces)

-Rotate board
